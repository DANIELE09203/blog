HTML Basics

Struttura Base HTML

<!DOCTYPE html> <html lang="it"> <head> <meta charset="UTF-8"> <title>Il Mio Sito</title> </head> <body> <h1>Hello World!</h1> </body> </html>

Elementi Base

Gli elementi HTML sono i blocchi di costruzione di ogni pagina web. Ogni elemento è definito da un tag di apertura e chiusura.

Attributi

Gli attributi forniscono informazioni aggiuntive sugli elementi e sono sempre specificati nel tag di apertura.

Struttura

Un documento HTML è strutturato in una gerarchia ad albero, con elementi annidati all'interno di altri elementi.

Examples

Form HTML

<form action="submit.php" method="POST"> <label for="name">Nome:</label> <input type="text" id="name" name="name"> <button type="submit">Invia</button> </form>
